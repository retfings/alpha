## 1. 测试概述

### 1.1 测试目标

验证股票筛选器前端功能的正确性、可用性和用户体验，确保：
- 指标说明问号提示正常显示
- 指标筛选操作（>, =, <, 范围）功能正确
- 权重排序功能正常工作
- 指标启用/禁用切换响应正确
- 响应式布局在不同设备上正常显示
- 主流浏览器兼容性良好

### 1.2 测试范围

| 功能模块 | 测试重点 | 优先级 |
|---------|---------|--------|
| 指标说明提示 | 问号 hover 显示 tooltip | P0 |
| 指标筛选操作 | 8 种操作符切换、值输入 | P0 |
| 权重排序 | 滑块拖动、自动平衡 | P1 |
| 启用/禁用切换 | Toggle 开关状态 | P1 |
| 响应式布局 | 断点切换、布局重排 | P2 |
| 浏览器兼容 | Chrome、Firefox、Safari、Edge | P2 |

### 1.3 测试类型

- ✅ 功能测试
- ✅ UI/UX 测试
- ✅ 兼容性测试
- ✅ 探索性测试
